Number Guessing Game Rules
1. Objective:
The goal is to guess the secret number correctly to earn points.
2. Scoring:
Each correct guess earns you +1 point.
Each incorrect guess deducts -1 point.
3. Winning Condition:
The game is won when the player reaches 10 points.
4. Secret Number:
The secret number is randomly generated.
The secret number changes to a new random number each time the player makes a correct guess.
5. Gameplay:
The player inputs their guess into the game.
Click the "Submit" button to check the guess.
The game will display whether the guess was correct or incorrect.
The player's score will be updated based on the result of their guess.
6. Game Reset:
The game can be reset to start over, clearing the current score and generating a new secret number.
7. Feedback:
After each guess, the game will provide feedback on whether the guess was correct or not.
The current score will be displayed after each guess.
8. User Interface:
A "Submit" button to check the guess.
A "Again" button to restart the game and clear the score.
